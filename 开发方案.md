# 🎋 DragonBoatFestival（端午节活动插件）开发方案

## 一、项目定位

插件名称：DragonBoatFestival

适用版本：

* Spigot 1.12.2
* Paper 1.12.2

开发语言：

* Java 8

兼容插件：

* Vault（可选）
* PlayerPoints（可选）
* PlaceholderAPI（可选）

设计目标：

* 独立运行
* 无需依赖菜单插件
* 配置简单
* 低性能占用
* 适合生存服、轻RPG服

---

# 二、核心玩法

获取方式：

# 八、性能设计

* 所有数据缓存内存
* 定时异步保存
* 玩家退出自动保存
* 不使用 NMS
* 不依赖数据库

预计性能消耗：

在线100人以下：

TPS影响接近0

适合长期活动服使用。

# 十、精品版升级方案

目标：把插件从“基础活动”升级为“玩家不用问、服主好配置、活动有目标感”的精品节日插件。

## 1. 玩家体验升级

### 活动中心主菜单

玩家输入：

```text
/duanwu
```

打开 GUI：

```text
🎋 端午活动中心
├─ 我的状态：积分、材料、签到、竞速状态
├─ 制作粽子：普通/豪华粽子 GUI 制作
├─ 活动商店：积分兑换奖励
├─ 每日签到：点击领取每日奖励
├─ 龙舟竞速：参加竞速、查看排行榜
├─ 幸运摸鱼：查看钓鱼奖励说明
└─ 新手指南：一步步告诉玩家怎么玩
```

### 新手引导

首次进入服务器时，如果活动开启，延迟发送小白引导：

```text
欢迎参加端午活动！
第一步：挖矿、破坏树叶、收作物、打怪、钓鱼获得材料。
第二步：输入 /duanwu 打开活动中心。
第三步：点击“制作粽子”获得积分。
第四步：点击“活动商店”兑换奖励。
```

同时提供：

```text
/duanwu guide
```

展示完整游玩闭环和常见问题。

### 玩家状态页

提供：

```text
/duanwu status
```

显示：

* 节日积分
* 背包内端午材料数量
* 今日是否已签到
* 龙舟竞速是否已开始/赛道是否可用
* 下一步推荐操作

## 2. 活动目标升级

### 个人目标

记录玩家：

* 制作普通粽子数量
* 制作豪华粽子数量
* 钓鱼中奖次数
* 商店兑换次数
* 签到天数

根据目标发放阶段奖励。

### 全服任务系统

配置全服目标：

```yaml
server-goals:
  make-zongzi:
    target: 500
    commands:
      - 'broadcast 端午全服任务完成！'
```

所有玩家共同推进任务，完成后全服发奖。

## 3. 玩法深度升级

### BOSS 活动

管理员可开启端午 BOSS：

```text
/duanwu boss spawn
```

BOSS 掉落积分、材料、礼包命令奖励。

### 龙舟竞速增强

增强内容：

* 加入后传送到起点
* 倒计时开始
* 支持退出比赛
* 支持最佳成绩提示
* 支持个人名次提示

## 4. 兼容增强

### PlaceholderAPI

如果服务器安装 PlaceholderAPI，注册变量：

```text
%duanwu_points%
%duanwu_rice%
%duanwu_leaf%
%duanwu_jujube%
%duanwu_meat%
%duanwu_signed_today%
```

### 可选经济兼容

保留独立积分，同时允许商店奖励命令对接 Vault、PlayerPoints 或其他插件命令。

## 5. 配置与文档升级

* Wiki 分为玩家入门版、管理员配置版、常见问题、活动测试清单。
* 所有 GUI 标题、物品、文案、命令奖励尽量可配置。
* 默认配置可直接上线，服主不改也能玩。
